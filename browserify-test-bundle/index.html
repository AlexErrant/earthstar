Earthstar - Browser test bundle

Open the browser console to see the test running.

This HTML file runs browserify-test-bundle/bundle.js in your browser. This bundle only contains one of the Earthstar test files because I couldn't figure out how to combine them all into a single bundle. Choose which test file you want to run in package.json in the browserify-for-karma command.

How to run

You can just drag this HTML file into a browser, or you can serve it using something like: 

npx serve

To regenerate the bundle file, run: 

npm run build && npm run browserify-for-karma

You can also run headless on the command line with either of these two commands, which also do the build and bundling steps for you: 

npm run test-in-browser-with-karma

npm run test-in-browser-run
